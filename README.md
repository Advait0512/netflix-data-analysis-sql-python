# 🎬 Netflix Data Analysis (SQL + Python)

## 📌 Overview
Analyzed Netflix dataset using SQL and Python to uncover content trends, country-wise distribution, and growth patterns with meaningful insights.

## 🚀 Features
- Complex SQL queries (Joins, CTEs)
- Data cleaning and preprocessing using Pandas
- Visualization of trends using Matplotlib
- Analysis of content types, countries, and release trends

## 🛠️ Tech Stack
- Python
- SQL
- Pandas
- Matplotlib

## 📂 Project Structure
- `analysis.py` – Data analysis script
- `queries.sql` – SQL queries used
- `dataset.csv` – Netflix dataset

## ▶️ How to Run
1. Load dataset into SQL database
2. Run SQL queries from `queries.sql`
3. Run Python script:
   ```bash
   python analysis.py
